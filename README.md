# Beginner MERN Projects 
In this repo I am going to develop some projects which are for beginner level , from which I will develop a complete understanding of CRUD operations, basic routing, and state management in React with Node.js/Express backends connected to MongoDB

## To-Do List App:
 Build an app for adding, editing, deleting, and marking tasks complete, implementing user authentication for persistence.​

## Personal Portfolio Website:
 Create a responsive site showcasing projects with a contact form, using React components and MongoDB for dynamic content.​

## Weather App: 
Fetch real-time weather data via API, display forecasts, and save favorites in MongoDB.
